## ğŸ’» Local Development

### View Locally

1. Clone this repository:
```bash
git clone https://github.com/YOUR_USERNAME/edubarrier-website.git
cd edubarrier-website
```

2. Open `index.html` in your browser:
```bash
# On Mac
open index.html

# On Windows
start index.html

# On Linux
xdg-open index.html
```

Or use a local server:
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (with http-server)
npx http-server
```

Then visit `http://localhost:8000`

## ğŸ”§ Making Changes

### Updating Content

1. Edit the HTML files directly
2. Commit your changes:
```bash
git add .
git commit -m "Updated homepage content"
git push
```

3. Vercel will automatically deploy (30 seconds)

### Changing Colors

Edit CSS variables in `styles.css`:
```css
:root {
    --color-teal: #0D9488;
    --color-emerald: #10B981;
    --color-indigo: #4F46E5;
    --color-purple: #9333EA;
}
```

### Updating Forms

The contact form in `contact.html` uses client-side validation. To make it functional:

1. Set up a form backend (Formspree, Netlify Forms, etc.)
2. Update the form action in `contact.html`
3. Configure email notifications
